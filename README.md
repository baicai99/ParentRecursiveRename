# 父级递归命名脚本

[中文](README.md) | [English](README_ENGLISH.md)

#### 作用

父级递归命名是一种命名规则，通常用于给文件、文件夹或变量等命名，其中较长的名称包含了更短名称的一部分。这种命名方式让我们能够更清晰地组织和标识事物，就像家谱中的家族命名一样。

#### 效果

~~~
- 家庭照片
  - 家庭照片_假日
    - 家庭照片_假日_2022_夏季假日
    - 家庭照片_假日_2023_冬季假日
  - 家庭照片_婚礼
  - 家庭照片_生日
~~~
